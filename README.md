## 工作流程

1. 字幕从 vtt 转换为 srt

2. 修正 srt 字幕

3. 机翻中文，生成 *.zh.srt

4. 人工校对翻译

## 文件说明

- auto-sub - 从 youtube 自动生成的字幕文件

- preview - 对字幕转换格式、去重后，生成的字幕文件
